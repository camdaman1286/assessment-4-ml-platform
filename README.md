# Assessment 4 — ML Platform

## Overview
Multi-service ML platform deployed on EKS.

## Phases
1. Infrastructure (Terraform + EKS)
2. Services deployed to EKS with probes
3. Namespaces and resource controls
4. CI/CD pipeline
5. React dashboard
6. Documentation and architecture diagram

## Structure
- `infrastructure/terraform` — EKS cluster provisioning
- `infrastructure/k8s` — Kubernetes manifests
- `services/` — Application services
- `docs/` — Architecture and runbooks
